# Allatmenhely nyilvantarto rendszer - Python

Ez a mappa a `oop_gyakorlo_feladat.md` feladat Python megoldasat tartalmazza.

## Fajlok

- `main.py`: a program belepesi pontja
- `allat.py`: ososztaly
- `kutya.py`: kutya osztaly
- `macska.py`: macska osztaly
- `fajlkezelo.py`: fajlkezelo osztaly
- `allatok.txt`: a beolvasott adatok

## Mukodes

A program:

- beolvassa az `allatok.txt` fajlt
- letrehozza az allat objektumokat
- kiirja az osszes allatot
- kulon megjeleniti az orokbefogadhato allatokat
- statisztikat keszit
- ev alapjan szuri a bekerult allatokat
- kulon fajlba menti az orokbefogadhato allatokat

## Futtatas

Lepj be ebbe a mappaba, majd futtasd:

```powershell
python main.py
```

A program beker egy evet a szureshez.
