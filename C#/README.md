# C# programok

Mivel a C# a Microsoft nyelve és arra lett készítve, hogy projekt alapon készítsünk alkalmazást (új projektet hozunk létre folyton), teljesen felesleges lenne a teljes (fordított kód, dll fájlok stb) programot feltölteni, ezért csak a .cs fájlokat (azaz a c# kódokat) fogom feltölteni.

## Használata

Megnyitod a fájlt, bemásolod a saját projektedbe (figyelve a namespace-re).

A fájlt átnevezem arra a dátumra, amikor óránk volt, s azzal dolgoztunk.
